=== configure, make, install

autoreconf::
+
[source, shell]
----
autoreconf -fi
----
https://manpages.ubuntu.com/manpages/latest/man1/autoreconf-dickey.1.html

configure::
+
[source, shell]
----
./configure
----

make::
+
[source, shell]
----
make
----

install::
+
[source, shell]
----
sudo make install
----

run::
+
[source, shell]
----
/usr/local/bin/alephone ~/Games/Marathon
----
